# Backend Simplificado (Recomendado)

Para trabajar con una arquitectura de microservicios basica, usa solo la version desde cero:

- `microservicios-basico/api-gateway`
- `microservicios-basico/usuarios`
- `microservicios-basico/cursos`
- `microservicios-basico/temas`
- `microservicios-basico/comentarios`
- `microservicios-basico/chatbot`

## Comando unico

Desde `backend/microservicios-basico`:

```bash
npm install
npm run install:all
npm run dev
```

O desde la raiz del proyecto:

```bash
npm run install:backend
npm run dev:backend
```

## Que pasa con las otras carpetas de backend

Las demas carpetas pertenecen a una version mas amplia del proyecto. Puedes ignorarlas para esta etapa y concentrarte solo en `microservicios-basico`.
